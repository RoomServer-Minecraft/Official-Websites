# クリスマス大作戦！（大遅刻）

## クリスマス大作戦とは？

このイベントは、冬休み期間に合わせてサーバーを盛り上げ、プレイヤー同士や運営との交流を深めることを目的とした大交流会です。\
「最近サーバーで誰も話していない」という現状を打破し、運営とプレイヤーが一緒に雑談したり遊んだりして活性化させることを目指しています。また、新規プレイヤーの方にサーバーを知ってもらうきっかけ作りや、サーバーの性能テストとしての側面も持っています

### 🗓️ イベントのスケジュールと内容&#x20;

12月28日（日） 13:00 〜 16:00 の3時間にわたり、以下の3部構成で開催されます。

1. 【第1部】<mark style="color:$primary;">**アスレチック大会 🧗‍♂️**</mark> 景色を楽しみながら、のんびり進めるコースに挑戦します。
2. 【第2部】<mark style="color:$primary;">**ブロックかくれんぼ 🙈**</mark> ブロックに化けて隠れるゲームです。全員が平等に遊べるよう、数回に分けて実施されます。
3. 【第3部】<mark style="color:$primary;">**運営 vs プレイヤー！マンハント（鬼ごっこ） 🏃‍♂️💨**</mark>  サバイバル形式での真剣勝負です！運営チームをプレイヤー全員で追いかけます（※当日の状況により開催）。
